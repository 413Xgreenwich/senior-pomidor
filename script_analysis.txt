Скрипт ищет все ссылки на заданном сайте

1. Скрипт принимает URL сайта в переменную url
2. Скрипт подключается к сайту посредством метода requests.get
3. Исходный код сайта (HTML) записывается в переменную html
4. Все ссылки ищутся в переменной html с помощью регулярного выращения и записываются в переменную links в виде списка кортежей
5. Ссылки выводятся по одной циклом, выбирая первый объект кортежа (второй объект - протокол, http(s) или ftp(s))

Пример использования - тестирование ссылок на веб-странице. Можно далее автоматизировать запросы по этим ссылкам и проверять их доступность, возвращая статус коды 200/404/500
